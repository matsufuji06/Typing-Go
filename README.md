### Typing-Go/概要  
表示された文字を入力し、正確さとかかった時間を競うタイピングゲームです。Go言語の練習のために作ってみました！

### 遊び方  
コンソールにて、git clone https://github.com/matsufuji06/Typing-Go でクローンしていただき、  
go run typing.goでスタートできます。
